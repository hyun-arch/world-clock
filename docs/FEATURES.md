# 🕐 World Clock · 세계 시계 — 전체 기능 정리

전 세계 동료·파트너와 컨퍼런스 콜을 잡을 때, **한국 시간(KST) 기준으로 상대방의 현지 시각과
업무 가능 여부를 한눈에 판단**하기 위한 세계 시계 가젯입니다.
하나의 UI 코어를 **① 데스크톱 가젯(Electron)** 과 **② 크롬 익스텐션** 두 가지 형태로 제공합니다.

---

## 1. 핵심 기능

| 기능 | 설명 |
|------|------|
| **기본 시계 4종** | 서울(기준) · 뉴욕 · 샌프란시스코 · 파리(프랑스)가 처음부터 표시됩니다. |
| **도시 추가/삭제** | 80여 개 주요 도시를 검색해 추가·삭제. 국가명/도시명(한글·영문) 모두 검색됩니다. |
| **한국(서울) 기준 고정** | 서울은 항상 맨 위에 **"기준"** 배지와 함께 고정되어 삭제할 수 없습니다. |
| **한국 대비 시차 표시** | 각 도시 카드에 `KST −14h`, `KST −7h`처럼 한국과의 시차를 함께 표시합니다. |
| **업무시간 색상 표시** | 카드 왼쪽 색 막대로 상대 도시의 현재 상태를 표시 — 🟢 업무시간(9–18) · 🟡 이른/늦은 시간(7–9·18–21) · ⚪ 야간·휴식. 주말 업무시간은 자동으로 🟡로 낮춰 표시합니다. |
| **회의 시간 슬라이더** | 한국 시간을 −12h~+12h로 밀어보면 **모든 도시 시각이 동시에 바뀌고** 업무시간 색상도 다시 계산됩니다. 모두에게 적당한 시간대를 즉시 탐색. "지금" 버튼으로 현재 시각 복귀. |
| **12/24시간 전환** | 상단 `24h/12h` 칩으로 즉시 전환. 12시간 표기 시 AM/PM 표시. |
| **날짜·요일 표시** | 각 카드에 월·일·요일(한글) 표시. 한국과 **날짜가 다른 도시**는 날짜를 노란색으로 강조(날짜변경선 판단에 유용). |
| **낮/밤 아이콘** | 각 도시가 낮(☀️)인지 밤(🌙)인지 아이콘으로 표시. |

## 2. 디자인

- **다크 글래스(글래스모피즘)** 테마 — 반투명 유리 질감 + 흰 글씨로 어떤 배경 위에서도 가독성이 높습니다.
- **컴팩트한 카드형 레이아웃** — 좁은 폭(약 340px)으로 화면 한쪽에 두어도 방해되지 않습니다.
- **드래그 정렬** — 카드를 끌어 순서를 바꿀 수 있습니다(서울 기준 카드는 항상 맨 위 고정).
- 업무시간(초록)·주의(노랑)·야간(회색)의 **의미 있는 색상 코딩**으로 직관적입니다.

---

## 3. 데스크톱 가젯(Electron) 전용 기능

> 예전 윈도우 가젯처럼 **다른 모든 창 위에 떠 있는** 진짜 데스크톱 위젯입니다.

- **항상 위에 표시(Always-on-top)** — 엑셀·아웃룩·브라우저 등 어떤 창 위에도 계속 보입니다. 📌 버튼으로 on/off.
- **테두리 없는 이동식 창** — 상단 바를 잡고 원하는 위치로 드래그. 위치·크기를 기억합니다.
- **창 투명도 조절** — 40~100%로 조절해 배경과 자연스럽게 어우러지게 할 수 있습니다.
- **트레이 상주** — 닫아도 종료되지 않고 시스템 트레이로 숨겨져 상시 대기합니다.
- **Windows 시작 시 자동 실행** — 설정 또는 트레이 메뉴에서 켜면 부팅 시 자동으로 떠 있습니다.
- **단일 인스턴스** — 중복 실행 시 기존 창을 앞으로 가져옵니다.

## 4. 크롬 익스텐션 전용 기능

> 크롬만 있으면 **설치 프로그램 없이** 어느 PC에서든 동일한 시계를 사용할 수 있습니다.

- **새 탭 화면** — 새 탭을 열 때마다 세계 시계가 표시됩니다.
- **사이드 패널** — 툴바 아이콘을 누르면 브라우저 오른쪽에 시계가 **고정 패널**로 열려, 작업 중에도 계속 확인할 수 있습니다.
- **설정 동기화** — 추가한 도시·표기 설정이 `chrome.storage`에 저장되어 새 탭/사이드 패널 간에 공유됩니다.

> 참고: 브라우저 확장은 보안상 **브라우저 밖(다른 앱) 위에는 뜰 수 없습니다.** 다른 창 위에 항상 띄우려면 데스크톱 가젯(Electron)을 사용하세요. 그래서 두 형태를 모두 제공합니다.

---

## 5. 기술적 특징

- **자동 서머타임(DST) 처리** — 모든 시차 계산은 브라우저 내장 `Intl` + IANA 타임존으로 실시간 계산합니다. 수동 오프셋 표가 없어 서머타임 전환·법령 변경에도 **항상 정확**합니다. (뉴욕: 여름 −13h / 겨울 −14h 자동 반영, 인도 −3:30 등 30분 단위 시차도 지원)
- **단일 코어(shared/)** — UI 로직을 한 벌만 작성하고 두 형태에서 공유합니다. 유지보수가 간단합니다.
- **의존성 최소** — UI는 순수 HTML/CSS/JavaScript(프레임워크·번들러 없음). 로직은 Node에서 단위 테스트로 검증(20개 통과).
- **개인정보 안전** — 모든 데이터는 로컬에만 저장, 외부 전송·네트워크 요청이 전혀 없습니다.

## 6. 폴더 구조

```
International clock/
├─ shared/            # UI 코어 (단일 소스): 시계 로직·화면·도시 데이터
├─ electron/          # 데스크톱 가젯 래퍼 (main/preload/package.json)
├─ chrome-extension/  # 크롬 익스텐션 (manifest/background + shared 복사본)
├─ build/             # 빌드 스크립트: sync(공유 복사)·테스트·미리보기 서버
├─ assets/            # 앱 아이콘(png/ico) + 생성 스크립트
└─ docs/              # 기능정리·사용 매뉴얼·발표자료
```
